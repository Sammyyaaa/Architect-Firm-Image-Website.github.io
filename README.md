# 建築師事務所形象網站

建築設計公司形象展示網站，包含作品集、服務介紹、建材批售商店等功能。

---

## 技術架構

### 框架與套件

| 套件                    | 版本   | 用途                                             |
| ----------------------- | ------ | ------------------------------------------------ |
| Bootstrap               | 5      | UI 框架、RWD、導覽列、Modal、Offcanvas、Carousel |
| Bootstrap Icons         | 1.10.3 | 圖示（CDN）                                      |
| AOS (Animate On Scroll) | -      | 滾動進場動畫（CDN）                              |
| jQuery                  | 2.2.3  | DOM 操作                                         |
| Popper.js               | -      | 工具提示定位                                     |

---

## 頁面結構

| 頁面           | 說明                               |
| -------------- | ---------------------------------- |
| `index.html`   | 首頁，包含視差滾動與公司介紹       |
| `About.html`   | 關於我們，公司理念與團隊介紹       |
| `Project.html` | 經典作品，圖片牆點擊放大瀏覽       |
| `Service.html` | 服務內容，建築／室內／景觀規劃說明 |
| `Shop.html`    | 建材批售，商品瀏覽與購物車功能     |
| `Contact.html` | 聯絡諮詢，表單與地圖               |

---

## 功能特色

### Project 頁面

- 四欄瀑布流圖片牆
- 點擊任一圖片開啟 overlay 大圖展示
- 圖片自動限制在視窗高度內，長形圖片等比縮放

### Shop 頁面

- 商品分類左側導覽欄（可展開收合）
- 點擊商品卡片開啟購買視窗，顯示商品圖片、名稱、售價
- 數量輸入自動計算總金額
- 確認送出後加入購物車
- 右側浮動購物車 icon，徽章顯示目前商品種類數
- 點擊購物車 icon 展開側邊清單，可查看明細與移除商品
